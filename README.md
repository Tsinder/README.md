# Core Concept

Every physical event leaves traces in multiple domains:

- Optical (photons, reflections, scattering)
- Acoustic (vibrations, resonance)
- Electromagnetic (EM fields, RF signals)
- Thermal (heat gradients)
- Structural (material deformation)

UPRS attempts to reconstruct past events by:

1. Detecting residual signals  
2. Modeling their propagation  
3. Reconstructing probable historical states
